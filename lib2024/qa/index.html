LIB2024- QA

  1. Qus: What are the three componets of Web?

    Ans: The three components of the Web are the url or Uniformn Resouce Locator, http or HyperText Transfer Protocol, and html or Hyper Text Markup Language.

  2. Qus: Define version control and give one example.

    Ans: Version control is a system where changes to files and codes can be tracked over time. An example of version control is updated versions of a software application like MS Word version 16.5.

  3. Qus: What tag is used to create the title of a web page? Give an example.

    Ans: The tag used to create the title of a web page is <title></title>.

  4. Qus: What tag is used to create a paragraph in html. Give an example.

    Ans: The tag used to create a paragraph in html is <p></p>.

  5. Qus: Identify the odd one from these tags: <span>, <p>, <a>, <i> and state the reason.

    Ans: The odd one from these tags is <p> because it's a block-level element. The other tags are inline elements.

  6. Qus: What are the tags used to create a navigation list in HTML?

    Ans: The <nav>, <ul> and <li> tags are used to create a navigation list in HTML.

  7. Qus: What element is used to create images? Name its attributes.

    Ans: The <img> element is used to create images. Its attributes are src and alt.

  8. Qus: Identify the odd one from these tags: <table>, <tr>, <b>, <td> and state the reason.

    Ans: The <b> is the odd one from these tags as it is a tag to format text as bold, while the others are tags used to create tables.

  9. Qus: Give an example of color picker input.

    Ans: The <input type="color"> creates a color picker for the input. For example: <label for="favcolor">Select your favorite color:</label> <input type="color" id="favcolor" name="favcolor">

  10. Qus: What is the use of name and value attributes? Give two examples.

    Ans: The name attribute specifies a name for an HTML element. For a form element, the name attribute is used as reference when the data is submitted. The input value attribute specifies an initial value for an input field. For <li> elements, the value attribute sets the value of the list item (for ordered lists). The next list items will increment from that value.

  11. Qus: Give an example of inline style.

    Ans: An example of an inline style is <p style="color:red;">This is a paragraph.</p>.

  12. Qus: How would you link an external style sheet? Give an example.

    Ans: The <link> tag is used to link to an external style sheet. For example: <link rel="stylesheet" type="text/css" href="css/style.css">

  13. Qus: Create CSS block to center an element both horizontally and vertically that has position:absolute and has width:200px and height:200px.

    Ans: div{ bottom: 0; height: 200px; left: 0; margin: auto; position: absolute; top: 0; right: 0; width: 200px; }

  14. Qus: Create CSS keyframes block called "spin" that rotates an element.

    Ans: @keyframes spin { from { transform: rotate(0deg); } to { transform: rotate(359deg); } }

  15. Qus: Create CSS animation block to rotate an element that has the following properties: class "rotate-me" It rotates 5 times at intervals of 0.5s with "linear" transition using animation keyframes "spin"

    Ans: .rotate-me { animation: spin .5s 5 linear; } @keyframes spin { from { transform: rotate(0deg); } to { transform: rotate(359deg); } }

  16. Qus: How would you include create an alert with string "I love JavaScript" using JavaScript?

    Ans: <script> function myFunction() { alert("I love JavaScript"); } </script>

  17. Qus: How would you fade out an element with id "fade-me" after a delay of 300ms using JQuery?

    Ans: $("#fade-me").fadeOut(300);

  18. Qus: How would you select all elements with class "my-class" using Javascript and Jquery?

    Ans: $( ".my-class" )

  19. Qus: Give an example of input field that takes phone number with format "123-456-7890"?

    Ans: <input type="tel" id="phone" name="phone" placeholder="123-456-7890" pattern="[0-9]{3}-[0-9]{3}-[0-9]{4}" required>

  20. Qus: Among HTML, CSS, Javascript, JQuery, what language did you like the best? State the reason.

    Ans: I preferred HTML, as it seemed easier than the other languages.
