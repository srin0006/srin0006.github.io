  1. What are the three components of the Web?

  2. Define version control and give one example.
    3. Version control is a system where changes to files and codes can be tracked over time. An example of version control is updated versions of a software application like MS Word version 16.5.

  3. What tag is used to create the title of a web page? Give an example.

  4. What tag is used to create a paragraph in html. Give an example.
    5. The tag used to create a paragraph in html is <p></p>.
